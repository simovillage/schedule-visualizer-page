# schedule-visualizer-pages

GitHub Pages 公開専用の最小構成です。

このフォルダの中身だけを、新しい GitHub リポジトリにアップロードしてください。

## 含まれるファイル

- `index.html`
- `styles.css`
- `app.js`
- `.nojekyll`

## 公開手順

1. GitHub で新しい公開リポジトリを作る
2. このフォルダの中身だけをそのリポジトリに入れる
3. GitHub の `Settings` → `Pages` を開く
4. `Deploy from a branch` を選ぶ
5. `main` と `/ (root)` を選ぶ
6. 数分待つ

公開 URL は次の形になります。

`https://<your-user-name>.github.io/<repository-name>/`

## 補足

この構成は静的ファイルだけなので、スマホからそのまま使えます。
